Giới thiệu:
Sử dụng thư viện Math để tính toán nâng cao.
    
Các hàm và cú pháp phổ biến:
- Math.sqrt(x), Math.pow(a,b)
- Math.max(a,b), Math.min(a,b)
- Math.random() // số ngẫu nhiên [0,1)
- Math.min(a, b)
- Math.abs(a)
- v.v

Thư viện cần import:
Không cần import (có sẵn).

Bài tập thực hành:
1. Tính căn bậc hai của một số.
2. Tìm số lớn nhất trong 3 số.
2. Sinh 5 số ngẫu nhiên từ 1–100.
