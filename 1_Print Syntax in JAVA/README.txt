Giới thiệu:
In dữ liệu ra màn hình bằng System.out trong Java.

Các hàm và cú pháp phổ biến:
- System.out.print("chuỗi");
- System.out.println("chuỗi"); // In kèm xuống dòng
- System.out.printf("%.2f", value); // Định dạng số/chữ

Thư viện cần import:
Không cần import (có sẵn trong java.lang).

Bài tập thực hành:
1. In ra 'Hello, World!'.
2. In ra số Pi với 2 chữ số thập phân.
2. In ra tên và tuổi của bạn trên 2 dòng.
