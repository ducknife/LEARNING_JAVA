Giới thiệu:
Xem README.txt để học về chủ đề này trong Java.

Các hàm và cú pháp phổ biến:
- Khai báo một mảng: int[] arr;
- Khởi tạo một mảng: int[] arr = {1, 2, 3, 4, 5};
hoặc chưa biết giá trị int[] arr = new int[Số phần tử];

Thư viện cần import:
Không cần import.

Bài tập thực hành:
1. Viết 1-2 ví dụ minh họa.
2. Làm 1 bài tập tự nghĩ ra.
2. Viết ghi chú kiến thức quan trọng.
