Giới thiệu:
Khai báo biến, kiểu dữ liệu cơ bản và kiểu tham chiếu trong Java.

Các hàm và cú pháp phổ biến:
- int x = 10; double y = 3.14; boolean flag = true;
- String name = "Java";
- final int CONST = 100; // Hằng số
- lưu ý 1: khi khởi tạo giá trị float, ta cần để chữ f ở cuối, vì mặc định nó sẽ tạo ra double.
- lưu ý 2: khi khởi tạo giá trị long, nếu giá trị vượt quá int max thì phải thêm chữ l ở cuối.

Thư viện cần import:
Không cần import (dùng java.lang).

Bài tập thực hành:
1. Tạo 3 biến: tên (String), tuổi (int), GPA (double) và in ra.
2. Khai báo hằng số số Pi và in ra.
2. Thử thay đổi giá trị biến và in lại.
